## 快速排序

思路：快速排序是先将一个元素排好序，然后再将剩下的元素排好序

```js
function sort(nums: number[], lo: number, hi: number) {
    if (lo >= hi) return 
    // 对nums[lo...hi]切分
    const p = partition(nums, lo, hi)
    sort(nums, lo, p-1)
    sort(nums, p + 1, hi)
}
```
